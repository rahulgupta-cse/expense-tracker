A simple and user-friendly Expense Tracker built using Python.
This project allows users to manage daily expenses efficiently using a command-line interface.

 Features

 Add expenses with:

Date (auto-generated)

Expense name

Category (Food, Travel, etc.)

Amount

 View all expenses in a clean format

 Delete expenses by name

 Data stored permanently in a CSV file

 Beginner-friendly & easy to use

 Technologies Used

Python

CSV (File Handling)

Datetime Module
